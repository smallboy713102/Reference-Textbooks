    \hypertarget{gradient-descent}{%
\section{Gradient Descent}\label{gradient-descent}}

Scikit-learn has a gradient descent regression model
\href{https://scikit-learn.org/stable/modules/generated/sklearn.linear_model.SGDRegressor.html\#examples-using-sklearn-linear-model-sgdregressor}{sklearn.linear\_model.SGDRegressor}.
Like your previous implementation of gradient descent, this model
performs best with normalized inputs.
\href{https://scikit-learn.org/stable/modules/generated/sklearn.preprocessing.StandardScaler.html\#sklearn.preprocessing.StandardScaler}{sklearn.preprocessing.StandardScaler}
will perform z-score normalization as in a previous lab. Here it is
referred to as `standard score'.
